# fmg-timesheets

Analysis of my timesheets under the former time keeping system at FMG.

## Usage

Store PDF-formatted timesheets locally, ina `data` folder.

```
mkdir data
cp path/to/timesheets/*.pdf data/
```

Run the program.

```
make run
```
